## Technologies/libraries that were used

- Nest.js;
- MongoDB;
- Mongoose;
- class-transformer;
- class-validator;

# What was implemented

* All CRUD operation: Get, Post, Patch, Delete;
* Integration with DB;
* Filter by genre and rating;
* Pagination;

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
