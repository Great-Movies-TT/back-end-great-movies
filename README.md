## Technologies/libraries that were used

- Nest.js;
- MongoDB;
- Mongoose;
- class-transformer;
- class-validator;

And `Insomnia` for api requests testing;

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
